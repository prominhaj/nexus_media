## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

-   Node.js
-   npm or yarn

### Installation

1. Clone the repo:
    ```sh
    git clone https://github.com/prominhaj/nexus_media.git
    ```
2. Install NPM packages:
    ```sh
    npm install
    # or
    yarn install
    ```
3. Run the development server:
    ```sh
    npm run dev
    # or
    yarn dev
    ```
